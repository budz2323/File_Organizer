# File_Organizer CLI Tool

This Project is a python commant-line toolthat organizes files in a directory by file extensions.

## Features
    - Organizes into extension-based folders
    - Safe dry-run mode (no files will be moved)
    - Automatic renaming to prevent over writing
    - Logging of all actions and errors

## Usage
In the command line 

python file_organizer.py target_dirctory 

python file_organizer.py target_dirctory --dry-run (to preview the changes that will be made)
